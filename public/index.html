Setup Instructions

  1. Go to Google Cloud Console
  2. Create a new project (or select an existing one)
  3. Enable the Google Sheets API under APIs & Services
  4. Go to Credentials → Create Credentials → Service Account
  5. Give it a name, then click Done
  6. Click the service account → Keys tab → Add Key → JSON
  7. Download the JSON key file
  8. Create a Google Sheet and share it with the service account email (with Editor access)
  9. Copy the Sheet ID from the URL: docs.google.com/spreadsheets/d/{SHEET_ID}/edit
